これは今では更新されておらずMCversion1.12.2で止まっているGulliver-Rebornを1.18.2で使えるようにしようとしているものです。
最終的には名前を含めて変更する予定ですが初めてMODの変更及び制作をするためある程度完了するまで元のMODの名前のままにしています(コードの中ではすでに変更している箇所もあります)
現在は必須依存ライブラリであるArtemisLibの1.18.2対応を行っているためこちらの変更作業は中断しています。
ArtemisLibのフォークリポジトリはありますが、そちらの方は参考にするためにフォークしており、実際に作っているのはgithubにまだ乗せていないのでこのプロジェクトに協力してくれる人がいるのならgithubに追加しようと考えています。

This is an attempt to make Gulliver-Reborn, which has not been updated and is stuck at MC version 1.12.2, usable with version 1.18.2.
I plan to eventually change the name as well, but since this is my first time modifying and creating a mod, I'm keeping the original name until I'm more or less done (I've already changed some parts of the code).
Currently, we are working on making ArtemisLib, a required dependent library, compatible with version 1.18.2, so work on this change has been suspended.
There is a forked repository of ArtemisLib, but I forked that one for reference purposes, and the actual work I am doing has not yet been put on GitHub, so if there are people who are willing to cooperate with this project, I would like to add it to GitHub.
